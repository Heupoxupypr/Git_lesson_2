## Lists

Для создания ненумерованного списка необходимо выделить пункты звездочкой (*) Например:
* Элемент 1
* Элемент 2
* Элемент 3

Для создания нумерованного списка необходимо просто пронумеровать пункты. Например:

1. Первый пункт
2. Второй пункт
3. Третий пункт
